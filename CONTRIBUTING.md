# Contributing

Want to help? Great. Here's how to get started.

## Setting Up Locally

```bash
git clone https://github.com/Ajay150313/sre-agent
cd sre-agent
go mod tidy
go test ./...
go run ./cmd/sre-agent/main.go
```

## What We Need

Bug fixes, new features, better tests. Documentation improvements. Examples of how you're using it.

## Making Changes

1. Fork the repo
2. Create a branch with your feature: git checkout -b feature/cool-feature
3. Make your changes
4. Test it: go test ./...
5. Commit: git commit -m "feat: add cool feature"
6. Push and open a PR

## Code Style

Just run gofmt on your code. Keep commits small and focused.

## Questions?

Open an issue or start a discussion. We're here to help.

Thanks for contributing.
